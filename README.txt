This application is a Restaurant Picker built using jQuery Mobile, HTML and CSS as front-end technology and Django as the back-end technology. 

The front-end code has been generated by Stéphanie Walter under this jQuery Mobile tutorial where you can find the specifications of the application: 

http://www.noupe.com/tutorial/jquery-mobile-tutorial-creating-a-restaurant-picker-web-app.html  

I, Yaiza Bailen, have developed the back-end code using Django and I have written a basic Django tutorial:

http://yaizabailen.com/django-tutorial-creating-the-back-end-for-the-restaurant-picker-app/
